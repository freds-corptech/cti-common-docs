The CommonApiExtensions type exposes the following members.

Methods
  NameDescription
AddDefaultApiServices
Registers default API services. Optionally, you can call the other methods in this class to register the services individually.
ConfigureAuthentication
Registers services related to authentication.
ConfigureAuthorization
Registers services related to authorization. Uses PermissionPolicyProvider and PermissionAuthorizationHandler.
ConfigureSwagger
Registers services related to Swagger documentation with default options defined in ConfigureSwaggerOptions
ConfigureVersioning
Registers services related to API versioning with default options.
EnableSwagger
Enables Swagger API documentation with common options.
